环境搭建

1.下载Golang:The Go Programming Language

2.配置镜像（编辑系统环境变量）。

变量 值
GO111MODULE on
GOPROXY https://goproxy.cn

3.go version可查看安装Go版本。

4.配置GOPATH（GOPATH:使用绝对路径提供项目的工作目录,项目代码放在GOPATH的src目录）

变量 值
GOPATH D:\Code\go
GOROOT Golang安装路径(一般安装自动配置)

GOPATH:

  • bin:编译生成的可执行文件
  • pkg:编译生成的归档文件
  • src:源代码

6.go env：查看Go环境信息。

2.go mod init [Project] 创建对应mod文件。

3.新建.go文件，按照VSCode提示安装对应工具gotools(可能需要代理)。

Goland

1.GoLand: A Clever IDE to Go by JetBrains

2.新建项目

3.Add Configuration

  • 名称：配置信息的名称，可自定义，也可使用系统默认的值。
  • Run kind：设置为Directory。
  • Directory：设置main包所在的目录（不能为空）。
  • Output directory：设置编译生成的可执行文件的存放目录（可以为空，为空时默认不生成可执行文件）。
  • Working directory：设置程序的运行目录（可以与Directory设置相同，不能为空）。
