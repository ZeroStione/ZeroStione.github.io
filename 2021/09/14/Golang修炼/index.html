Golang修炼

发表于 2021-09-14 更新于 2021-09-22 分类于 Programming Language

Go的语法接近C语言，但对于变量的声明有所不同。Go支持垃圾回收功能。Go的并行计算模型以Tony Hoare的通信顺序进程（CSP/Communicating Sequential Process）为基础，Go具有这个模型的特征，比如通道传输。与C++相比，Go并不包括如枚举、异常处理、继承、泛型、断言、虚函数等功能，但增加了切片(Slice)型、并发、管道、垃圾回收功能、接口等特性的语言级支持。Go 2.0版本将支持泛型。Go原生提供了关联数组（哈希表（Hashes）或字典（Dictionaries））。

通过《The Go Programming Language》进行Golang的入门学习。

入门

  1. Go是编译型语言(.go)，Go的工具链将程序的源文件转换成机器相关的原生二进制指令。
  2. Go原生支持Unicode。
  3. Go代码使用包来组织(类似于库和模块)，包由一个或多个.go源文件组成，放在一个文件夹中，该文件夹名字描述了包的作用。
  4. fmt包中的函数用来格式化输出和扫描输入。
    • Println是fmt的一个基本输出函数，结尾使用一个换行符。（以ln(line)结尾的格式化函数，遵循Println的方式。)
    • 默认情况下，Printf不会换行。（以字母f(format)结尾的格式化函数，采用fmt.Printf的格式化准则。）
  5. main包用来定义一个独立的可执行程序，而不是库。
  6. main包中main函数总是程序开始执行的地方。
  7. Go中所有索引使用半开区间，即包含第一个索引，不包含最后一个索引。
  8. Go语言不允许使用无用的局部变量（local variables）。

Golang撰写风格

  1. 在语句或声明后不需要使用分号结尾（;），除非有多个语句或声明出现在同一行（跟在特定符号的换行符被转换为分号）。
  2. 大括号（{）不能够换行放置（{必须和关键词func在同一行）。
  3. if判断式和for循环不需要小括号。
  4. 使用tab做排版。
  5. 尽量使用短小的命名。
  6. 驼峰式命名（优先使用大小写分隔，如QuoteRuneToASCII和parseRequestLine）。

命令行参数

  1. 命令行参数以os包中Args名字的变量提供访问，在os包外面，使用os.Args。

  2. os.Args是一个字符串slice。

  3. os.Args的第一个元素：os.Args[0]，是命令本身的名字,其它的元素则是程序启动时传给它的参数。

  1. 程序导入多个包:
    • 用括号把写成列表形式(习惯)
    • 分开写成独立的import声明
  2. gofmt工具格式化时按照字母顺序对包名排序。
  3. 变量会在声明时直接初始化。如果变量没有显式初始化，则被隐式地赋予其类型的零值（zero value），数值类型是0，字符串类型是空字符串""。

1.for循环

  1. 左大括号必须和post语句在同一行。
  2. initalization如果存在，必须是一条简单语句（simple statement），即短变量声明、自增语句、赋值语句或函数调用。
  3. condition是一个布尔表达式（boolean expression），其值在每次循环迭代开始时判断。
  4. post语句在循环体执行结束后执行。
  1. 每次循环迭代，range 产生一对值；索引以及在该索引处的元素值。range的语法要求要处理元素，必须处理索引。
  2. Go语言中用空标识符（blank identifier），即_（下划线）。空标识符可用于在任何语法需要变量名但程序逻辑不需要的时候（如：在循环里丢弃不需要的循环索引并保留元素值）。

处理输入循环

  1. if语句条件两边不加括号，但是主体部分需要加。
  2. map 存储键/值（key/value）的集合，对集合元素提供常数时间的存、取或测试操作。（Go的map类似于Java语言中的HashMap，Python语言中的dict，通常使用hash实现）
    • 键可以是任意类型，只要其值能用==运算符比较，最常见的例子是字符串；
    • 值可以是任意类型。
  3. bufio包通常用于IO操作。Scanner类型是该包最有用的特性之一，它读取输入并将其拆成行或单词。
  4. Printf函数的首个参数是个格式字符串，指定后续参数被如何格式化。

获取URL

  1. http.Get函数是创建HTTP请求的函数，如果获取过程没有出错，那么访问的请求结果会存在resp结构体中。resp的Body字段包括一个服务器响应的可读数据流。

并发获取多个URL

  1. goroutine是一种函数的并发执行方式，channel用来在goroutine之间进行参数传递。main函数本身运行在一个goroutine中，而go <function>表示创建一个新的goroutine，并在这个goroutine中执行该函数。
  2. 当一个goroutine尝试在一个channel上做send或者receive操作时，这个goroutine会阻塞在调用处，直到另一个goroutine从这个channel里接收或者写入值，这样两个goroutine才会继续执行channel操作之后的逻辑。

Web服务

  1. 请求是http.Request类型的对象（结构体struct），这个对象中包含了请求中的一系列相关字段，其中包括URL。当请求到达服务器时，这个请求会被传给handler函数处理，这个函数会将/hello这个路径从请求的URL中解析出来，然后将其发送到响应中。

命名

命名必须以字母（Unicode字母）或下划线开头，后面可以跟任意数量的字母、数字或下划线，并且区分大小写。

  • 关键字
break default func interface select
case defer go map struct
chan else goto package switch
const fallthrough if range type
continue for import return var

  • 预声明:内建的常量、类型和函数

    • 常量

      true false iota nil

    • 类型

      int int8 int6 int32 int64
      uint uint8 uint16 uint32 uint64
      uintptr float32 float64 complex128 complex64
      bool byte rune string error

    • 函数

      make len cap new append
      copy delete complex real imag
      panic recover close
  1. 如果在函数内部定义，那么就只在函数内部有效。如果是在函数外部声明，那么将在当前包的所有文件中都可以访问。
  2. 名字的开头字母的大小写决定其在包外的可见性。如果名称以大写字母开头（在函数外部定义的包级名字），那么它将是导出的，即可以被外部的包访问，如fmt包的Printf函数就是导出的，可以在fmt包外部访问。
  3. 包名本身一般使用小写字母。
  4. 通常名称的作用域越大，生命周期越长，就使用越长且更有意义的名称。

声明

  • 声明语句定义了程序的各种实体对象以及部分或全部的属性。

  • Go语言主要有四种类型的声明语句：

    • var:变量
    • const:常量
    • type:类型
    • func:函数

  • 源文件中以包的声明语句开始，说明该源文件是属于哪个包。package语句之后是import语句导入依赖的其它包，然后是包级别的类型、变量、常量、函数的声明，不区分顺序（函数内部的名字必须先声明之后才能使用）。

  1. 在包一级声明语句声明的名字（如常量boilingF）不仅在其声明所在的源文件中访问，而且对于同一包里面的所有源文件都可访问。相比之下，局部声明的名字就只能在函数内部范围被访问。
  2. 函数的声明由一个函数名字、参数列表（由函数的调用者提供参数变量的具体值）、一个可选的返回值列表和包含函数定义的函数体组成。如果函数没有返回值，那么返回值列表是省略的。函数的第一个语句开始执行，直到遇到第一个return返回语句，如果没有返回语句则执行到函数末尾，然后返回到函数调用者。

变量

var声明语句可以创建一个特定类型的变量，然后给变量附加一个名字，并且设置变量的初始值。

type或= expression可以省略其中一个:

  • 如果省略类型，那么将根据初始化表达式来推导变量的类型信息。
  • 如果省略初始化表达式，那么将用零值初始化该变量。
    • 数值类型变量:0。
    • 布尔类型变量:false。
    • 字符串类型:空字符串""。
    • 接口或引用类型（slice、指针、map、chan和函数）变量：nil。
    • 数组或结构体等聚合类型：其元素或字段的零值。
  • 零值初始化机制确保每个声明的变量总是有一个良好定义的值，因此在Go语言中不存在未初始化的变量。
  1. 初始化表达式可以是字面量或任意的表达式。在包级别声明的变量会在main函数执行前完成初始化，局部变量将在声明语句被执行到的时候完成初始化。
  2. 一组变量可以通过调用一个函数，由函数返回的多个返回值初始化。
