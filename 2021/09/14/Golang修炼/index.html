Golang修炼

发表于 2021-09-14 更新于 2021-09-15 分类于 Programming Language

Go的语法接近C语言，但对于变量的声明有所不同。Go支持垃圾回收功能。Go的并行计算模型以Tony Hoare的通信顺序进程（CSP/Communicating Sequential Process）为基础，Go具有这个模型的特征，比如通道传输。与C++相比，Go并不包括如枚举、异常处理、继承、泛型、断言、虚函数等功能，但增加了切片(Slice)型、并发、管道、垃圾回收功能、接口等特性的语言级支持。Go 2.0版本将支持泛型。Go原生提供了关联数组（哈希表（Hashes）或字典（Dictionaries））。

通过《The Go Programming Language》进行Golang的入门学习。

入门

  1. Go是编译型语言，Go的工具链将程序的源文件转换成机器相关的原生二进制指令。
  2. Go原生支持Unicode。
  3. Go代码使用包来组织(类似于库和模块)，包由一个或多个.go源文件组成，放在一个文件夹中，该文件夹名字描述了包的作用。
  4. fmt包中的函数用来格式化输出和扫描输入。
    • Println是fmt的一个基本输出函数，结尾使用一个换行符。（以ln(line)结尾的格式化函数，遵循Println的方式。)
    • 默认情况下，Printf不会换行。（以字母f(format)结尾的格式化函数，采用fmt.Printf的格式化准则。）
  5. main包用来定义一个独立的可执行程序，而不是库。
  6. main包中main函数总是程序开始执行的地方。
  7. Go中所有索引使用半开区间，即包含第一个索引，不包含最后一个索引。
  8. Go语言不允许使用无用的局部变量（local variables）。

Golang撰写风格

  1. 在语句或声明后不需要使用分号结尾（;），除非有多个语句或声明出现在同一行（跟在特定符号的换行符被转换为分号）。
  2. 大括号（{）不能够换行放置（{必须和关键词func在同一行）。
  3. if判断式和for循环不需要小括号。
  4. 使用tab做排版。

命令行参数

  1. 命令行参数以os包中Args名字的变量提供访问，在os包外面，使用os.Args。

  2. os.Args是一个字符串slice。

  3. os.Args的第一个元素：os.Args[0]，是命令本身的名字,其它的元素则是程序启动时传给它的参数。

1.echo.go

  1. 程序导入多个包:
    • 用括号把写成列表形式(习惯)
    • 分开写成独立的import声明
  2. gofmt工具格式化时按照字母顺序对包名排序。
  3. 变量会在声明时直接初始化。如果变量没有显式初始化，则被隐式地赋予其类型的零值（zero value），数值类型是0，字符串类型是空字符串""。

2.for循环

  1. 左大括号必须和post语句在同一行。
  2. initalization如果存在，必须是一条简单语句（simple statement），即短变量声明、自增语句、赋值语句或函数调用。
  3. condition是一个布尔表达式（boolean expression），其值在每次循环迭代开始时判断。
  4. post语句在循环体执行结束后执行。

3.echo2.go

  1. 每次循环迭代，range 产生一对值；索引以及在该索引处的元素值。range的语法要求要处理元素，必须处理索引。
  2. Go语言中用空标识符（blank identifier），即_（下划线）。空标识符可用于在任何语法需要变量名但程序逻辑不需要的时候（如：在循环里丢弃不需要的循环索引并保留元素值）。

1.dup1.go

  1. if语句条件两边不加括号，但是主体部分需要加。
  2. map 存储键/值（key/value）的集合，对集合元素提供常数时间的存、取或测试操作。（Go的map类似于Java语言中的HashMap，Python语言中的dict，通常使用hash实现）

%T 变量的类型
