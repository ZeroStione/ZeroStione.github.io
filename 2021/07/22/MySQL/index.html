MySQL基础

1.优点

  • 实现数据持久化
  • 使用完整的管理系统统一管理，易于查询

2.相关概念

DB 数据库（database）

DBMS 数据库管理系统（Database Management System）

MySQL DB2 Oracle Sqlsever

  1. 基于共享文件系统的DBMS Access
  2. 基于客户机—服务器的DBMS MySQL Oracle SqlSever

SQL 结构化查询语言(Structured Query Language)

1.几乎所有DBMS都支持SQL

2.简单易学

3.SQL特点

  1. 数据放到表中，表再放到库中。
  2. 一个数据库中可以有多个表，每个表都有一个名字，用来标识自己，表名具有唯一性。
  3. 表具有一些特性，这些特性定义了数据在表中如何存储。
  4. 表由列组成，也可称为字段。
  5. 表中的数据是按行存储的。

4.MySQL特点

MySQL

  1. MySQL数据库隶属于MySQLab公司，总部位于瑞典，后被oracle收购。

  2. 优点

    1. 成本较低，开放源代码，一般可免费使用

    2. 性能高

    3. 操作简单：容易安装和使用

5.MySQL服务启动与关闭

  • CMD PS中root用户需提权

net start [mysqlname]

net stop [mysqlname]

6.MySQL服务客户端的登录与退出

环境变量:C:\Program Files\MySQL\MySQL Server 8.0\bin

  • MySQL 8.0 Command Line Client

    • Enter password
    • exit

  • Command Line

    • mysql -h localhost -P 3306 -u root -p
    • exit
    • mysql –version/-V

7.常见命令

Commands end with ; or \g.

  1. show databases 查看当前所有数据库
  1. **create database [database_name] **
  1. use [database_name] 打开指定数据库

mysql> use test;
Database changed

  1. show tables 查看当前库的所有表
  1. show tables from [database_name] 查看指定库的所有表
  1. select database() 查看所在库

  1. create table teainfo(code int ,sta varchar(10))

    创建表

    create table [table_name](

    ​ 列名 列类型,

    ​ 列名 列类型,

    ​ ……

    );

  1. drop table [table_name] 删除指定表
  1. desc [table_name] 查看表结构
  1. select * from [table_name]
  1. insert into stuinfo (id,name) values(1,’john’)
  1. update stuinfo set name=’jack’ where id=1
  1. delete from stuinfo where id=1
  1. select version();

8.常见规范

  1. 不区分大小写（建议关键字大写，列名、表名小写）

  2. 每条命令建议使用分号;结尾

  3. 命令可进行缩进和换行

  4. 注释

    • 单行注释
    • 多行注释
